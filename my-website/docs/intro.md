# Adali Clothing Dokumentáció

Üdvözlünk az Adali Clothing webáruház hivatalos dokumentációjában! Ez a dokumentáció részletesen bemutatja a webáruház működését, funkcióit és technikai megvalósítását mind a felhasználók, mind a fejlesztők számára.

## A webáruházról

Az Adali Clothing egy modern, teljes funkcionalitású webáruház, amely lehetővé teszi:

- Ruházati termékek böngészését és vásárlását
- Felhasználói regisztrációt és bejelentkezést
- Saját termékek feltöltését eladásra
- Értékelések és vélemények írását
- Kedvezményes kuponok használatát
- Rendelések kezelését és nyomon követését

## Technológiai stack

Az Adali Clothing webáruház a következő technológiákat használja:

### Backend
- Node.js
- Express.js
- MySQL adatbázis
- JSON Web Token (JWT) alapú hitelesítés

### Frontend
- React.js
- Material UI komponenskönyvtár
- Redux állapotkezelés
- Axios HTTP kliens

## Rendszerkövetelmények

- Node.js 14.x vagy újabb
- MySQL 8.0 vagy újabb
- Modern webböngésző (Chrome, Firefox, Safari, Edge)
